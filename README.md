<h1>Silent Chemical Alarm</h1>

<p style="text-size=12pt;">Built with Node.JS and AWS IoT Enterprise Button.
  Integration with the intercom will be built with Ruby on a Virtual Machine running Windows Server. </p>
  
To install, use a AWS IoT button that is initialzied through the AWS IoT Application on Android or iOS.

Login to the AWS console and go to the Lambda functions.  Select the text message blueprint and upload the .JS code located in this repo.

Replace the placeholder numbers in the .JS file with the numbers you need to alert. Verify that you are using the correct country code.

More updates will be added when the work on the Virtual Machine begins.
